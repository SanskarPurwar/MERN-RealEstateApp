# HeavenNest - Real Estate Platform

Welcome to **HeavenNest**, a user-friendly real estate platform where users can explore properties for rent and sale, create wishlists, and communicate seamlessly with property owners through real-time chat.

[Visit the Live Project](https://heavennest.onrender.com/)

---

## Features

1. **Property Listings**: 
   - View properties for rent or sale with detailed descriptions, images, and prices.

2. **Search, Filter, and Sort**: 
   - Search properties by keywords.
   - Filter by type, furnished status, price range, and discount availability.
   - Sort listings by creation date or other criteria.

3. **Wishlist**: 
   - Save properties to a personal wishlist for easy reference.

4. **Real-Time Chat**:
   - Directly connect with property owners via real-time messaging powered by **Socket.io**.

5. **Pagination**:
   - Efficiently browse listings with paginated views for a smoother user experience.

6. **User-Friendly UI**:
   - Sleek and responsive design built using **ReactJS** and **TailwindCSS**.

---

## Tech Stack

- **Frontend**: ReactJS, TailwindCSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: Firebase, JWT
- **Real-Time Features**: Socket.io
- **State Management**: Redux Toolkit
